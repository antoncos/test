
## Памятка по запуску проекта

- Клонирование проекта
- Копирование `.env.example` в `.env`
- `composer install`
- `npm install`
- Генерация ключа приложения: `php artisan key:generate`
- `docker-compose up`
- `php artisan migrate`
- `npm run build`

Порт по умолчанию: `8000`

> http://localhost:8000/
